# Hero Video

Place your Hero video file here as:

/public/videos/hero.mp4

Requirements:

- Format: MP4 (H. 264) - widest browser support
- Resolution: 1920x1080 minimum, 4K preferred
- Duration: 10-30 seconds recommended
- No audio track needed (video is muted)
- File size: aim for under 15MB using compression

The video will:

1. Auto-play muted on page load
2. Play once and stop on the last frame
3. Trigger the navbar + company name reveal when it ends

Optional: also provide hero.webm for better compression in chrome/firefox
